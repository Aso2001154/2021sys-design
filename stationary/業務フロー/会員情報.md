```uml
@startuml

opt 会員情報
ユーザー -> Webサーバー : 会員情報をクリック
Webサーバー -> DBサーバー : 会員情報を照会
DBサーバー -> Webサーバー : 照会結果
Webサーバー -> ユーザー : 会員情報表示

opt 会員情報更新
ユーザー -> Webサーバー : 会員情報を入力し、updateボタンをクリック
Webサーバー -> DBサーバー : 会員情報を照会
DBサーバー -> DBサーバー : 更新処理
DBサーバー -> Webサーバー : 処理結果

alt 更新成功
Webサーバー -> ユーザー : 更新成功表示
else 更新失敗
Webサーバー -> ユーザー : 更新失敗表示
end
end
end

@enduml
```
