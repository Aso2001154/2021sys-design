```uml
@startuml

opt カートに商品を追加
ユーザー -> Webサーバー : カートに追加
Webサーバー -> DBサーバー : カートに追加された情報を渡す
DBサーバー -> DBサーバー : カート情報を保存
DBサーバー -> Webサーバー : 照会結果
Webサーバー -> ユーザー : カート情報を表示
end

opt 削除機能
ユーザー -> Webサーバー : カート内削除
Webサーバー -> DBサーバー : カートに追加された情報を削除
DBサーバー -> DBサーバー : カート情報を保存
Webサーバー -> ユーザー : 削除画面を表示し、カート内に戻る
end

@enduml
```
