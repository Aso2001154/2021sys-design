```uml
@startuml
opt カートに商品を追加
ユーザー -> Webサーバー : add to cartボタンをクリック
Webサーバー -> DBサーバー : カートに追加された情報を渡す
DBサーバー -> DBサーバー : カート情報を保存
DBサーバー -> Webサーバー : 照会結果
Webサーバー -> ユーザー : カートに追加完了を表示

opt 削除機能
ユーザー -> Webサーバー : 削除ボタンをクリック
Webサーバー -> DBサーバー : カートに追加された商品を削除
DBサーバー -> DBサーバー : カート情報を保存
Webサーバー -> ユーザー : 削除完了画面を表示し、カート内に戻る
end
end


@enduml
```
